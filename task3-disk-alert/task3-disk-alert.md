# Task 3 — Bash: Алерт при высоком использовании диска

## Описание
Скрипт проверяет занятие `/` диска и отправляет email, если занято более 85%. SMTP данные хранятся в `.env`.

## Структура
```
task3-disk-alert/
├── disk_alert.sh
├── .env
```

## Пример запуска
```bash
source .env
bash disk_alert.sh
```
